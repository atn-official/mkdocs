# Tailscale

Tailscale — это сервис для построения VPN-сети.
Данная сеть является безопасной, так как работает на основе протокола WireGuard.

??? node "Установка на linux"
    ??? node "Команда установки клиента"
        curl -fsSL https://tailscale.com/install.sh | sh
    ??? node "Команда для подключения клиента к Tailscale-сети"
        sudo tailscale up
        !!!node "" 
            При выполнении команды в терминале будет выведен url для авторизации данного хоста в tailscale-сети.
            Данный url можно открыть на любом устройстве.

??? note "Доп. функции"
    !!!node ""
        Exit node → Данная функция позволяет другим хостам выбрать хост на котором это включено, как свой gateway.
    !!!node ""
        Subnet routes → Данная функция позволяет анонсировать доступные хосту подсети в tailscale-сеть.
    !!!node ""
        Allow LAN access → Данная функция позволяет получить прямой доступ к локальной подсети в обход включённого Exit node.

??? note "Сценарий использования"    
    Сервер, на котором развёрнут docker-контейнер, подключён к Tailscale-сети. В данном контейнере производится администрирование данного сайта.
    Также к Tailscale-сети можно подключить ноутбук. Используя его, через IDE можно подключиться к контейнеру и производить администрирование данного сайта из любой сети.
